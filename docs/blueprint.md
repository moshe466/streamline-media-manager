# **App Name**: Streamline Media Manager

## Core Features:

- Admin Dashboard: Dashboard providing at-a-glance system status for admins.
- Client Dashboard: Client dashboard showing allowed streams and status.
- OTP Authentication: OTP email verification to ensure secure access for all users.
- Multi-Viewer Control (MCR): Multi-Viewer Control (MCR) display of live streams for admin, client, and viewers, permission based.
- Stream Filtering: Stream status filtering to quickly find and manage online/offline streams.
- Smart Alert System: Smart Alert tool that notifies admins of stream anomalies or server communication issues.
- Branding Customization: Customizable branding to upload logos for display on the admin and client interfaces, with a separate logo for offline streams.

## Style Guidelines:

- Primary color: Deep Blue (#1E3A8A) to convey professionalism and trust.
- Background color: Light gray (#F7F7FA) to ensure a clean, modern interface.
- Accent color: Vibrant cyan (#06BCC1) for interactive elements and key actions, drawing the user's attention.
- Headline font: 'Space Grotesk', a modern sans-serif, offering a computerized, scientific feel appropriate for the application.
- Body font: 'Inter', also sans-serif, giving a machined, objective, neutral appearance and ensuring readability.
- Use a set of consistent and clear icons, mainly outline style, to represent different functions and statuses throughout the platform.
- Implement a responsive and adaptive layout to ensure a seamless experience across various devices and screen sizes.
